🎬 Parrot API 批量處理測試

支援最多 50 張圖片的批量視頻生成

🚀 新功能特點

  • 支援50張圖片 - 從原本的10張提升到50張
  • 智能批量處理 - 可設定每批處理的任務數量
  • 可調整延遲 - 批次間延遲避免API限制
  • 實時進度監控 - 顯示當前批次和整體進度
  • 並行處理 - 批次內的任務並行執行
  • 錯誤處理 - 單個任務失敗不影響其他任務

📦 批量處理設置選項

批次大小選項：

  • 3 tasks per batch (推薦) - 適合穩定處理
  • 5 tasks per batch - 平衡速度和穩定性
  • 10 tasks per batch - 較快處理
  • Process all at once - 一次性處理所有任務

批次間延遲選項：

  • 2 seconds - 快速處理
  • 5 seconds (推薦) - 避免API限制
  • 10 seconds - 保守處理
  • 30 seconds - 最安全的選項
⚠️ 注意事項：
  • 建議使用較小的批次大小（3-5個任務）以避免API限制
  • 處理大量圖片時請耐心等待，系統會自動管理任務隊列
  • 如果遇到錯誤，可以嘗試增加批次間延遲時間
  • 確保有足夠的API配額來處理所有任務

🔧 使用步驟

  1. 啟動代理服務器：
    python parrot_proxy_server.py
  2. 在瀏覽器中打開：
    http://localhost:8000
  3. 配置API設置（Provider、Version、API Key）
  4. 選擇最多50張圖片
  5. 設定批量處理參數（批次大小、延遲時間）
  6. 輸入提示詞（可選）
  7. 點擊開始生成
  8. 觀察進度條和任務狀態
  9. 等待所有任務完成
  10. 使用一鍵下載功能獲取所有視頻
