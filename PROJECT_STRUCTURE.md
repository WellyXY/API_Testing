## üöÄ Quick Start

### Run Development Server
```bash
export PORT=8000
python3 parrot_proxy_server.py
```

Then visit: `http://localhost:8000/parrot_api_frontend.html`

### Run Batch Server
```bash
python3 start_batch_server.py
```
