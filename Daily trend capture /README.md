# Google趋势数据可视化

这是一个简单的前端页面，用于展示基于特定关键词的Google趋势数据。目前使用的是模拟数据，展示了以下内容：

1. 过去7天内关键词的全球趋势变化
2. 热门相关关键词
3. 搜索量最高的国家排名

## 功能特点

- 可视化展示Google趋势数据
- 支持自定义关键词搜索
- 响应式设计，适配各种设备
- 使用Chart.js绘制直观的图表

## 使用说明

1. 直接在浏览器中打开`index.html`文件
2. 默认关键词设置为"PixVerse"
3. 在搜索框中输入其他关键词并点击"搜索"按钮（或按回车键）可更新数据

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js (数据可视化库)

## 注意事项

- 目前使用的是模拟数据，而非真实的Google Trends API数据
- 要获取真实数据，需要实现后端API来访问Google Trends数据
