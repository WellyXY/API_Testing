# Chrome 视频下载器

这是一个Chrome浏览器扩展，可以帮助您从当前浏览的网页中下载视频文件。

## 功能特点

- 自动检测当前页面上的所有视频
- 支持多种视频格式：MP4, WebM, OGG, M3U8等
- 支持从多种标签中提取视频：video, source, iframe, object, embed
- 支持检测页面内嵌脚本中的视频链接
- **一键批量下载所有视频，无需二次确认**
- **自定义保存文件夹，方便分类管理**
- 显示视频类型和大小信息

## 安装方法

### 从Chrome网上应用店安装（推荐）

1. 访问[Chrome网上应用店链接]()
2. 点击"添加到Chrome"按钮

### 手动安装（开发模式）

1. 下载本扩展源代码并解压
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 启用右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"按钮
5. 选择解压后的扩展文件夹

## 使用方法

1. 在含有视频的网页上，点击工具栏中的扩展图标
2. 扩展会自动扫描页面上的所有视频
3. **输入自定义的文件夹名称**（默认为"VideoDownloads"）
4. 点击列表中视频旁边的"下载"按钮下载单个视频
5. 或者点击"一键下载所有视频"按钮批量下载所有视频，无需二次确认

所有视频将自动保存在浏览器下载文件夹中的您指定的目录下。扩展会记住上次使用的文件夹名称，方便下次使用。

## 注意事项

- 某些网站采用复杂的流媒体技术或DRM保护，可能无法下载其视频
- 对于流媒体视频（如m3u8格式），下载后可能需要使用其他工具转换格式
- 请尊重版权，不要下载未经授权的内容
- 本扩展仅用于个人学习和合法用途

## 隐私声明

- 本扩展不会收集或上传任何个人数据
- 所有操作都在您的浏览器本地完成
- 不需要任何服务器端支持
