# Pika积分系统需求说明

## 1. 新用户注册奖励

- 为新注册用户一次性发放20 Credits，永久有效
- 用户完成注册流程后立即发放
- 通过弹窗告知用户已获得积分奖励

## 2. 每日登录奖励

- 用户每日首次登录或注册时发放10 Credits
- 每日积分在UTC+0 00:00过期
- 判断逻辑：检查用户当日是否已领取，如已领取则不再发放
- 付费会员同样适用此规则

## 3. 产品界面修改

### 3.1 会员页面更新
- 显示用户当前积分，区分"永久积分"和"每日积分"
- 会员页面增加"积分明细"区块
- 增加积分获取渠道的说明文案

### 3.2 首次进入提示
- 新用户首次进入应用时，弹窗告知积分规则
- 重点说明每日登录奖励机制

### 3.3 每日登录提示
- 用户每日首次登录时，轻量级提示已获得当日积分奖励
- 采用Toast或轻量级通知形式，不打断用户操作流程

## 4. 推送通知功能

- 针对24小时未登录的用户发送推送通知
- 提醒用户领取每日积分，提高回访率
- 推送频率：最多每3天一次，避免打扰用户
- 推送时间：用户所在时区的上午10:00-12:00之间
